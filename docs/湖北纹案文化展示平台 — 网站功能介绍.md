# 湖北纹案文化展示平台 — 网站功能介绍

> **域名（暂定）**: HBpattern.husteread.com
> **定位**: 湖北传统纹绣文化的数字化展示、探索、创作与共享平台
> **文档版本**: v2.1
> **最后更新**: 2026-03-31

---

## 一、首页

网站的门面入口，让用户第一眼就感受到传统纹绣之美。

### 主要内容

- **视觉主横幅（Hero）**：以全屏高清纹绣大图 + 动态光影效果作为背景，配文化主题标语
- **每日精选**：由管理员精心挑选的"今日焦点纹案"，配简短介绍
- **热门排行榜**：按浏览量/点赞数/评论数排列的纹案榜单
- **快速搜索入口**：支持关键词搜索，一键跳转高级搜索
- **最新上传**：社区最近上传的纹案动态流
- **统计概览**：平台已收録纹案总数、覆盖地区数、注册用户数等

---

## 二、纹案画廊

平台的核心展示区域，以瀑布流或网格形式展示所有已收錄的纹案。

### 浏览方式

- **瀑布流模式**：类似 Pinterest，大小不一的卡片错落排列，视觉效果丰富
- **网格模式**：统一尺寸的卡片整齐排列，方便快速浏览

### 筛选与搜索

- **多维度筛选面板**：可按以下维度自由组合筛选
  - 所属时期（楚文化 / 汉代 / 明清 / 近现代 ...）
  - 所属地区（武汉 / 恩施 / 黄冈 / 荆州 ...）
  - 工艺技法（刺绣 / 织锦 / 蜡染 / 扎染 / 挑花 ...）
  - 主色调（红 / 蓝 / 金 / 黑 ...）
  - 标签分类（凤鸟纹 / 云纹 / 几何纹 / 花卉纹 ...）
  - 是否为 AI 生成
  - 版权类型（公有领域 / CC 协议 / 版权所有）
- **以图搜图**：上传一张纹案图片，系统自动找到视觉相似的纹案
- **颜色搜索**：点选色板颜色，搜索包含该主色调的纹案
- **地图联动搜索**：在地图页框选一片区域后，自动跳转画廊并筛选该范围内的纹案

### 纹案卡片信息

每个卡片展示：缩略图、纹案名称、所属时期、所属地区、点赞数，以及以下标识：
- 🤖 **AI 标记**：如果该纹案由 AI 生成，会有醒目的"AI 生成"标记
- ©️ **版权标识**：显示版权协议类型（如 CC-BY-NC-SA 等）

---

## 三、纹案详情页

点击任意纹案卡片即可进入详情页，全面了解该纹案的历史、文化和工艺信息。

### 页面内容

- **高清图片展示**：支持放大查看细节，多角度图片轮播
- **视频资料**：部分纹案配有制作工艺视频或历史纪录片段
- **文字介绍**
  - 纹案名称与简介
  - 详细历史背景（图文并茂，Markdown 富文本）
  - 所属时期与所属地区
  - 使用的工艺技法
  - 关联的非遗项目（如有）
- **色彩色板**：自动提取的纹案主色调展示，展现传统配色之美
- **关联纹案**：基于标签和特征推荐的"你可能也喜欢"相似纹案
- **演化关系**：该纹案与其他纹案的传承/衍生关系链接（如有）

### 用户互动

- **点赞**：登录用户可以为喜爱的纹案点赞
- **收藏**：将纹案加入自己的收藏夹，并可添加个人备注
- **评论区**：用户可以发表评论和讨论，支持回复嵌套
  - 所有评论均经过 AI 自动安全审核，确保社区氛围
- **举报**：如发现不当内容，可提交举报（不实信息/侵权/低质/冒犯性内容等）
- **分享**：生成分享链接

### 版权信息

详情页底部展示：
- 版权归属（公有领域 / 创作者姓名）
- 版权协议类型（如 CC-BY-NC-SA 4.0）
- 来源声明（原创拍摄 / 已授权使用 / 公共领域）
- 如为 AI 生成，标注 AI 模型版本

## 四、3D 文化地图

以交互式 3D 地图为核心的沉浸式探索页面，追溯每个纹案的地理渊源。

### 地图功能

- **中国地图全景**：初始展示中国地图全貌，所有纹案的位置以发光标记点呈现
- **无限缩放**：可以从全国视图自由缩放到街道级别，类似谷歌地图体验
- **拖拽探索**：自由拖拽浏览不同区域
- **地区边界**：展示湖北省各市县的行政区边界

### 交互效果

- **点击标记点**：
  1. 镜头自动平滑推进聚焦该区域（FlyTo 动画）
  2. 触发 **3D 光晕扩散特效**，以金色/朱红色光环从点击位置向外辐射
  3. 弹出该纹绣的简介浮窗（缩略图 + 名称 + 简介）
- **查看详情**：从浮窗一键跳转该纹案的完整详情页
- **查看周边**：展示同一区域的其他纹案
- **热力图**：以热力颜色展示哪些区域的纹案最为集中

### 地区文化介绍

点击某个行政区域时，可展示该地区的整体文化介绍，包括：
- 该地区的纹绣文化特色概述
- 该地区包含的非遗项目
- 该地区的代表性纹案列表

## 五、纹案演化时间线

可视化展示纹案从古至今的传承与演变脉络。

### 时间线视图

- 以横轴时间线形式，展示某类纹案（如凤鸟纹）从楚文化时期到现代的变迁历程
- 每个时间节点对应一个具体纹案，配缩略图和简要说明
- 点击节点可跳转该纹案详情页

### 关系图谱

- 以**力导向图**形式展示纹案之间的关联关系网络
- 关系类型包括：
  - 🔄 **演化自**：纹案 A 是由纹案 B 演变而来
  - 🔗 **受影响于**：纹案 A 的设计受到纹案 B 的影响
  - 🌿 **同源**：两个纹案拥有相同的文化源头
  - 📐 **变体**：纹案 A 是纹案 B 的地区变体或工艺变体
- 用户可以拖拽、缩放图谱，探索纹案之间的深层联系

## 六、AI 创作中心

利用 AI 技术进行纹绣图案创作和文创产品预览。

### AI 纹绣图案生成

- **文字描述生成图案**：用户输入描述（如"以凤鸟和祥云为主题，融合荆楚文化元素，红金配色"），AI 自动生成对应的纹绣图案
- **风格选择**：可选择预设风格模板（楚文化风 / 土家织锦风 / 汉绣风 / 现代简约风 等）
- **生成结果**：
  - 展示 AI 生成的高清纹案图片
  - 自动标注"AI 生成"标记
  - 用户可保存到自己的收藏夹，或提交到社区分享
- **用量限制**：
  - 普通用户每日 5 次
  - 贡献者等级用户每日 20 次
  - 专家及以上无限制

### 3D 文创产品预览器

- **选择纹案**：从收藏夹或画廊中选择任意纹案
- **选择产品类型**：
  - 🏺 杯子 / 马克杯
  - 👕 T恤 / 衣服
  - 🏁 旗帜
  - 🛏️ 被子 / 毯子
  - 🛋️ 抱枕
- **实时 3D 预览**：
  - 纹案自动贴到选定产品的 3D 模型表面
  - 用户可 360° 旋转、缩放查看效果
  - 实时调整纹案的位置、大小、角度
- **导出/分享**：截图保存当前效果，或生成分享链接

## 七、社区互动

用户可以在社区中分享、讨论和共创纹绣内容。

### 用户上传纹绣

- 注册用户可以上传自己收集或创作的纹绣图案
- 上传时**必须填写**：
  - 纹案名称和描述
  - 版权声明（原创拍摄 / 已授权使用 / 公共领域）
  - 如为 AI 生成，系统自动标注
- 上传的图片经过 AI 自动安全审核
- 审核通过后在画廊展示；如有疑问进入人工审核队列

### 评论与讨论

- 每个纹案详情页下方有独立的评论区
- 支持回复嵌套（可以回复别人的评论）
- 所有评论经 AI 安全审核后展示
- 如发表被拒绝的内容会收到通知告知原因

### 举报与反馈

- 用户可对纹案、评论或其他用户进行举报
- 举报理由：不实信息 / 版权侵权 / 低质内容 / 冒犯性内容 / 其他
- 举报提交后进入管理员处理队列
- 被举报者有权进行申诉

## 八、个人中心

登录用户的个人空间，管理自己的所有内容和互动。

### 我的主页

- **贡献统计**：上传纹案数、获得的总赞数、评论数
- **贡献者等级**：根据活跃程度分为四个等级
  - 🌱 新手 — 注册即获得
  - ⭐ 贡献者 — 上传 ≥ 10 个已审核纹案
  - 🏅 专家 — 获赞 ≥ 100 且通过专业审核
  - 👑 认证传承人 — 官方认证的非遗传承人
- **成就徽章**：展示获得的各类徽章
  - 「首次上传」「百赞纹案」「楚文化专家」「地图探索家」等

### 我的收藏夹

- 可创建多个收藏夹（如"我喜爱的凤鸟纹"、"研究素材"、"配色参考"）
- 每个收藏夹可设为**公开**（他人可通过链接查看）或**私有**
- 支持对收藏的每个纹案添加个人备注
- 可导出收藏夹为 PDF 或图片合集

### 通知中心

- 接收以下通知并管理已读/未读状态：
  - 💬 有人回复了你的评论
  - ✅ 你上传的纹案审核通过 / ❌ 被拒绝
  - 🏅 获得了新徽章
  - 📋 举报处理结果
  - 📢 系统公告
- 支持一键全部标记为已读

### 浏览历史

- 查看自己最近浏览过的纹案记录
- 方便找回之前看过但忘记收藏的内容

### 我的创作

- 查看自己使用 AI 生成的所有纹案作品
- 管理自己上传的纹案（编辑、删除）

### 个人数据管理

- 导出个人数据（下载自己的所有数据副本）
- 删除个人数据（注销账户并清除所有信息）

## 九、后台管理面板

仅管理员和审核员可访问的管理界面。

### 数据概览看板

全方位的平台运营数据统计：

| 统计类别 | 具体指标 |
|---------|---------|
| 流量概览 | 页面浏览量(PV)、独立访客(UV)、平均停留时长、跳出率 |
| 内容数据 | 纹案总数、本周新增、AI 生成占比、待审核积压量 |
| 用户数据 | 注册用户总数、日活/月活(DAU/MAU)、用户留存曲线 |
| 地图热力图 | 哪些地区的纹案最受关注 |
| AI 使用统计 | 生成总次数、用量趋势、热门 Prompt |
| 社区健康 | 评论量趋势、举报率、审核通过率 |
| API 使用 | 调用量、Top 调用者、错误率 |

### 内容审核管理

- 查看所有待审核的纹案和评论
- 一键通过 / 拒绝（填写拒绝理由）
- 将优秀纹案标记为"精华"推荐到首页

### 举报处理队列

- 查看所有未处理的举报
- 操作：确认违规并下架内容 / 驳回举报
- 处理被举报者的申诉请求

### 用户管理

- 查看用户列表和详细信息
- 调整用户角色（普通用户 / 审核员 / 管理员）
- 处理多次违规用户（限制发布权限等）

### 纹案管理

- 批量管理纹案的状态（上架/下架/精华）
- 管理纹案之间的关联关系（知识图谱编辑）
- 管理非遗项目关联

### API Key 管理

- 查看已发放的开放 API Key
- 审核新的 API Key 申请
- 管理各 Key 的调用配额和权限

## 十、开放 API 服务

面向开发者和研究机构的数据接口服务，让纹案数据可以被更广泛地利用。

### 功能

- 查询纹案列表（支持筛选、分页）
- 获取单个纹案详情
- 搜索纹案（关键词、地区、时期等）
- 获取地区列表和地区下的纹案
- 获取标签分类体系

### 使用方式

- 用户在个人中心申请 API Key
- 所有请求需携带 API Key 认证
- 速率限制：
  - 🆓 免费 Tier：100 次/小时
  - ⭐ 基础 Tier：1000 次/小时
  - 💎 高级 Tier：10000 次/小时

### 授权体系

- **学术研究**：免费使用
- **个人项目**：免费使用
- **商业用途**：需另行申请商用授权

## 十二、手机端适配

网站完整支持手机端浏览，所有功能在移动设备上均可正常使用：

- 响应式布局，自动适配各种屏幕尺寸
- 3D 地图支持手指捏合缩放和滑动拖拽
- 3D 文创预览器支持触摸旋转
- 可"添加到主屏幕"作为类 App 使用
- 已浏览过的纹案在弱网环境下仍可查看
