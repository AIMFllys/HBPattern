## 4. 共享组件分析

### 4.1 SiteHeader（82 行，Client Component）

- **Props**: `logoIcon`, `siteName`, `primaryColor`（cinnabar | gold）
- **功能**: Logo + 5 个导航链接 + 搜索框 + Dashboard 按钮 + 用户按钮
- **路由高亮**: ✅ 使用 `usePathname()` 判断激活态
- **问题①**: 动态 Tailwind 类 `text-${primaryColor}` → **可能被 Tree-shake**
- **问题②**: 搜索框纯视觉，无搜索功能
- **问题③**: 移动端无汉堡菜单（`hidden md:flex`），小屏导航完全消失

### 4.2 Icon（205 行，Server Component）

- **实现**: 30 个手写 inline SVG，通过 `name` prop 查找
- **问题①**: 与已安装的 `lucide-react` 功能完全重复
- **问题②**: 每次渲染都创建完整的 `iconComponents` 对象（性能浪费）
- **问题③**: 部分图标 SVG path 与名称不对应（如 `smart_toy` 用了 `person` 的 path）
- **问题④**: 未找到匹配时返回 `null`，无 fallback 提示
